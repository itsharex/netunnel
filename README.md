# 网跃通 (WangYueTong)

内网穿透工具，实现外网访问内网服务。

## 项目简介

网跃通是一款内网穿透工具，帮助用户将内网服务暴露到公网，实现远程访问。

- **目标用户**：需要远程访问内网服务的开发者、企业用户
- **核心功能**：内网穿透、端口映射、流量转发
- **部署状态**：规划中

## 技术栈

- **核心语言**：Go
- **服务端**：Go + Docker
- **客户端**：Go

## 端口规划（40060-40069）

| 端口 | 服务 | 环境 | 描述 |
|-----|------|------|------|
| 40060 | web | dev | 管理后台Web服务 |
| 40061 | api | dev | API接口服务 |
| 40062 | tunnel | dev | 隧道服务端口 |

## 部署指南

### 环境要求

- Docker 20.10+
- Go 1.20+

### 部署步骤

1. 克隆项目
2. 配置环境变量
3. 启动服务
